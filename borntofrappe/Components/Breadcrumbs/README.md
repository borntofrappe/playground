# Breadcrumbs

Navigation at the top of every page, highlighting the current position on the website and allowing to move back to previous routes.

## Layout

```code
borntofrappe / blog / blog-post

  ^^^^^^^    ^^^^^^^
  show       include
  by         a
  default    slash
```
